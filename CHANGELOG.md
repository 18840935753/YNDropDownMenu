# Installation & Update

YNDropDownMenu is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

`pod 'YNDropDownMenu'`

## 1.2.0 (2017-02-24)

* Available to call hideMenu in YNDropDownView
